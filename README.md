## 简介

- OceanMooc是一款开源在线教育平台，包含服务端、Android客户端、Web管理端。
- om-admin-vue工程是项目群中的Web管理端，他基于Webpack打包构建，使用Vue.js + Element开发，部署于Nginx中，实现前后端解耦。
